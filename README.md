# Streamlit
Running the Application
To start the Streamlit server and launch the app in your browser, run:
bash :  streamlit run app.py

#Grading Logic

The application uses the following default grading scale. 
You can modify this scale in app.py :
A: 90 - 100
B: 80 - 89
C: 70 - 79
D: 60 - 69
E: Below 60
